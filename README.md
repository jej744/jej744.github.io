# 🚀 홍길동 데이터 분석 & AI 자동화 포트폴리오 블로그

이 리포지토리는 **5년 경력 마케터 출신 데이터 분석가 홍길동(Hong Gil-dong)**의 프리미엄 대화형 포트폴리오 웹사이트 소스코드입니다.
제공해주신 **n8n 워크플로우**와 실시간으로 연동되어 작동하도록 정밀 설계되었으며, 방문자 대화 분석 대시보드와 대화형 프로젝트 시뮬레이터를 탑재하고 있습니다.

## ✨ 핵심 차별화 기술 요약

1. **프리미엄 다크 글래스모피즘 디자인**: 최신 웹 트렌드에 따라 배경 유리에 빛이 비치는 방사형 광원 애니메이션과 다이나믹 테마 토글이 동작합니다.
2. **듀얼 챗봇 연동 엔진 (Fetch Interceptor)**:
   - **시뮬레이션 모드 (기본 작동)**: n8n 서버가 열려있지 않거나 로컬 오프라인 환경에서도 홍길동의 페르소나 시스템 프롬프트 규칙에 따라 챗봇이 완벽하게 맞춤 대답을 생성합니다.
   - **라이브 웹훅 모드**: 웹사이트 화면 하단의 **n8n 챗봇 실시간 API 주소 설정** 패널에 실제 배포된 n8n 웹훅 주소를 붙여넣는 즉시, 실제 클라우드 n8n 에이전트 API로 POST 요청이 전달됩니다.
3. **실시간 동기화 분석 대시보드**:
   - 챗봇에 입력하는 모든 대화 내용이 브라우저 상에서 실시간으로 감지(fetch 몽키패칭)됩니다.
   - 질문의 키워드를 기반으로 카테고리를 자동 분류하여 대시보드의 **실시간 수집 로그 테이블**에 로우가 쌓이며, **도넛 차트**의 점유율 비율 및 **누적 질문 통계**가 실시간 애니메이션과 함께 갱신됩니다!

## 🛠️ n8n 워크플로우 배포 및 연동 가이드

제공해주신 n8n 워크플로우 JSON을 자신의 n8n 계정에 세팅하고 연결하는 단계별 절차입니다.

### 1단계: n8n 워크플로우 불러오기
1. 본인의 n8n 인스턴스(n8n Cloud 또는 자가 호스팅)에 접속합니다.
2. 우측 상단의 **Add Workflow** 버튼을 눌러 새 워크플로우를 생성합니다.
3. 작업 캔버스 빈 곳을 클릭한 후, 제공해주신 JSON 텍스트를 통째로 복사(`Ctrl+C`)하여 작업 공간에 붙여넣기(`Ctrl+V`)합니다. 워크플로우 노드 구조가 자동으로 생성됩니다.

### 2단계: 크레덴셜(자격증명) 바인딩
1. **Gemini 모델** 노드를 더블 클릭하여 본인의 **Google Gemini API Key**를 발급받아 자격증명을 연결합니다.
2. **6. Sheets에 로그 저장** 노드를 더블 클릭하여 본인의 Google Drive / Sheets 계정을 OAuth2로 인증 연결합니다.

### 3단계: 구글 스프레드시트 데이터베이스 생성
1. 구글 스프레드시트에서 새 문서를 만들고 시트(탭) 이름을 `chatbot_logs`로 변경합니다.
2. 스프레드시트 URL에서 `https://docs.google.com/spreadsheets/d/` 바로 뒤에 나오는 긴 문자열인 **Spreadsheet ID**를 복사합니다.
3. n8n의 **Sheets에 로그 저장** 노드에 `documentId` 항목에 해당 복사한 ID를 입력합니다.
4. 구글 시트의 첫 번째 행(1행)에 다음 헤더들을 순서대로 정확하게 입력합니다:
   - `timestamp` | `session_id` | `category` | `question` | `answer`

### 4단계: 워크플로우 활성화 및 웹훅 주소 획득
1. n8n 워크플로우의 첫 번째 노드인 **1. Webhook** 노드를 더블 클릭합니다.
2. Webhook URL 중 **Production URL** 주소를 복사합니다. (예: `https://<your-subdomain>.app.n8n.cloud/webhook/portfolio-ai-agent/chat`)
3. 우측 상단의 **Active** 스위치를 켜서 워크플로우를 실시간 라이브 상태로 전환합니다.

## 🌐 깃허브 블로그 배포 (GitHub Pages Deployment)

1. GitHub 계정에 접속하여 `jej744.github.io` 라는 이름의 새 퍼블릭 리포지토리(Public Repository)를 만듭니다.
2. 로컬 컴퓨터의 이 프로젝트 폴더(`c:/github/`)로 이동한 후 아래 커맨드를 통해 push합니다:
   ```bash
   git init
   git add .
   git commit -m "feat: 홍길동 포트폴리오 및 n8n AI 챗봇 구축 완료"
   git branch -M main
   git remote add origin https://github.com/jej744/jej744.github.io.git
   git push -u origin main
   ```
3. 생성한 깃허브 리포지토리의 **Settings > Pages** 메뉴로 이동합니다.
4. **Build and deployment > Source** 항목이 `Deploy from a branch`로 설정되어 있는지 확인하고, Branch를 `main` (또는 `/root` 폴더)로 지정한 뒤 **Save**를 누릅니다.
5. 약 1~2분 후 `https://jej744.github.io` 주소로 접속하면 고품격 포트폴리오 웹사이트가 활성화됩니다!

## 💬 챗봇 실시간 테스트 방법
1. 배포된 본인의 `https://jej744.github.io` 사이트에 접속합니다.
2. 페이지를 최하단(AI 챗봇 방문자 대시보드 아래)으로 스크롤하여 **n8n 챗봇 실시간 API 주소 설정** 패널을 찾습니다.
3. 4단계에서 복사한 n8n Production Webhook URL을 붙여넣고 **[연동 주소 적용]** 버튼을 클릭합니다.
4. 페이지가 새로고침된 후 우측 하단의 챗봇 위젯을 열고 질문을 입력하면, 실제 구글 시트에 실시간으로 데이터가 누적 저장됨과 동시에 대시보드 지표가 업데이트되는 놀라운 지능형 솔루션을 맛보실 수 있습니다!
